# WGUPS Routing Program
A package delivery routing system built for WGU's C950 Data Structures & Algorithms II.

## Features
- Custom hash table implementation
- Nearest neighbor routing algorithm with deadline-priority tiers
- Multi-truck delivery simulation
- CLI with colorama/tabulate
